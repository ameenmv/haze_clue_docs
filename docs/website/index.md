# Landing Website

The `Haze_clue_website` repository contains the marketing and informational web presence for the HazeClue platform. 

## Goal
To convert visitors into users by clearly explaining the benefits of combining cognitive training with tDCS and EEG hardware.

## Technology Stack
- **Framework:** Vue.js / Nuxt.js (or equivalent modern web framework).
- **Styling:** Custom CSS / Tailwind CSS, adhering to the deep purple (`#8B5CF6`) and dark mode aesthetic of the mobile application.

## Key Pages
1. **Home:** Value proposition, hero section, and hardware compatibility.
2. **Features:** Deep dive into the cognitive games and neurostimulation.
3. **Science:** Explanation of the research behind tDCS and EEG integration.
4. **Download:** Links to App Store and Google Play (mockups).

## SEO Best Practices
- Semantic HTML5 elements (`<header>`, `<main>`, `<article>`).
- Optimized meta descriptions and Open Graph tags.
- High-performance asset loading to maximize Lighthouse scores.
