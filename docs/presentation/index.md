# Presentation & Pitch

The `hazeclue-presentation` repository contains the assets, slide decks, and speaker notes for the final graduation project defense.

## Core Message
"HazeClue bridges the gap between hardware neurostimulation and software cognitive training, providing an accessible, AI-driven platform for mental performance enhancement."

## Slide Structure Guidelines
1. **The Problem:** Cognitive decline, stress, and the inaccessibility of clinical brain stimulation.
2. **The Solution:** HazeClue's integrated platform (App + tDCS/EEG + Smartwatch).
3. **Architecture:** High-level system design demonstrating the integration of Flutter, .NET Core, and Python AI models.
4. **Live Demo:** Walkthrough of the application, focusing on the seamless UX and the tDCS simulation session.
5. **Future Work:** Commercialization, clinical trials, and expanding the hardware ecosystem.
